1. In the plugin’s admin area go to Grid Accordion -> All Accordions, and click on the Import Accordion button.

2. Copy the content from the example’s text file and paste it in the modal window. Then, click on the Import button. Please note that images will not be imported, only URL’s of images.

3.  In the plugin’s admin area go to Grid Accordion -> Custom CSS & JS, and in the Custom CSS field copy the content from the custom-css.txt file.

That’s all.
